Objects & Properties

Objects are complex data structures that store collections of key-value pairs. They allow grouping of related data and functions.

Real-world Analogy

Think of an Object like a physical Car. A car has properties like color: "red" and brand: "Toyota". It also has methods (actions) like startEngine(). An object groups all these traits into one single entity.

Syntax Overview

Creating and accessing an object:

const user = {
  name: "John",
  age: 30,
  isAdmin: true
};
console.log(user.name); // "John"
