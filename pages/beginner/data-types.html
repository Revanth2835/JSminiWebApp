Basic Data Types

Data types specify the kind of data a variable holds. JavaScript is dynamically typed, meaning the same variable can hold different data types at different times.

Real-world Analogy

If variables are boxes, Data Types refer to what the object inside the box actually is. Is it a piece of text (String)? A math quantity (Number)? A True/False switch (Boolean)? Knowing the type tells you what you can do with it.

Syntax Overview

The core primitive data types in JavaScript:

let text = "Hello!"; // String
let count = 42; // Number
let isActive = true; // Boolean (true/false)
let notDefinedYet; // undefined
let emptyValue = null; // null
