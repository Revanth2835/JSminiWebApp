Conditionals

Conditionals allow your code to make decisions. They execute different blocks of code based on whether a specific condition evaluates to true or false.

Real-world Analogy

Conditionals are like a fork in a road or an intersection. IF the light is green, you drive. ELSE IF the light is yellow, you slow down. ELSE (meaning red), you stop.

Syntax Overview

The standard `if...else` statement:

if (condition) {
  // Runs if true
} else if (otherCondition) {
  // Runs if first is false and second is true
} else {
  // Runs if nothing else was true
}
