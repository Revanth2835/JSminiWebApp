Strings & Methods (API Reference)

A comprehensive guide to JavaScript Strings. Strings represent text, and JavaScript provides a rich set of built-in properties and methods to analyze, extract, and manipulate textual data easily. Note: In JavaScript, strings are immutable, meaning string methods always return a new string, rather than modifying the original.

Creating Strings & The .length Property

The `length` property returns the precise number of characters (including whitespaces) inside a given string.

🔍 Character Access & Codes

String.prototype.charAt() / at()

Returns: String

Returns the character precisely at the specified index. The newer `.at()` method allows negative integers to count backward from the end.

Syntax

  • str.charAt(index)
  • str.at(index) (Supports negative indices)

String.prototype.charCodeAt() / codePointAt()

Returns: Integer

Returns the UTF-16 code (an integer from 0 to 65535) or full Unicode code point representing the character at the specified index.

🔎 Searching & Matching

String.prototype.indexOf() / lastIndexOf()

Returns: Integer (Index)

Returns the index of the first (or last) occurrence of a substring. Returns -1 if not found.

Syntax

  • str.indexOf(searchString, position?)

String.prototype.includes() / startsWith() / endsWith()

Returns: Boolean

Convenience methods to quickly check if a string contains, starts with, or ends with a specific substring. Highly useful for conditionals.

String.prototype.match() / matchAll()

Returns: Array / Iterator

Retrieves the results of matching a string against a Regular Expression. matchAll is ideal for globally matching multiple capture groups.

String.prototype.search()

Returns: Integer (Index)

Executes a search for a match between a regular expression and this String. Returns the index of the first match, or `-1`.

🛠 Editing & Substrings

String.prototype.slice() / substring()

Returns: New String

Extracts a section of a string and returns it as a new string, without modifying the original. slice() supports negative indices to extract from the end!

Syntax

  • str.slice(indexStart, indexEnd?)

String.prototype.replace() / replaceAll()

Returns: New String

Replaces a pattern (string or RegExp) in the string with a replacement string.

String.prototype.toLowerCase() / toUpperCase()

Returns: New String

Returns the calling string value converted to lower or upper case. Crucial for case-insensitive string comparisons.

String.prototype.trim() / trimStart() / trimEnd()

Returns: New String

Removes whitespace from both ends, or just the start/end of a string. Excellent for cleaning up form inputs.

String.prototype.padStart() / padEnd()

Returns: New String

Pads the current string with another string until the resulting string reaches the given length.

String.prototype.split()

Returns: Array of Strings

Divides a String into an ordered list of substrings by splitting on a pattern, puts them into an Array, and returns the Array. The bridge between Strings and Arrays!

String.prototype.concat() / repeat()

Returns: New String

concat() combines string instances (though + or template literals are more common). repeat() constructs a new string containing a specified number of copies of the string.
