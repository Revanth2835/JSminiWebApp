Interview Based Questions

Output-based JavaScript questions are incredibly common in technical interviews. They test your knowledge of hoisting, closures, event loop, and tricky type coercions.

Real-world Analogy

These questions are like riddles. The engine doesn't always read code exactly how human intuition expects. Mastering these riddles proves you understand the deep gears of the JavaScript engine.

Syntax Overview

A classic scoping trick:

for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 100);
}
// OUTPUT: 3, 3, 3
// Explanation: 'var' is function-scoped. By the time the timeout runs, the loop has finished and 'i' is exactly 3.
