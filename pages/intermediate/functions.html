Functions & Arrow Functions

Functions are reusable blocks of code designed to perform a particular task. They prevent repetition and help organize code.

Real-world Analogy

A function is like a recipe. You give it inputs (ingredients), it follows a set of hidden steps (cooking), and it returns an output (a cake). You can reuse this same recipe anytime without rewriting the steps!

Syntax Overview

A classic function and an arrow function (ES6):

// Traditional
function add(a, b) {
  return a + b;
}

// Arrow Function
const addArrow = (a, b) => a + b;
