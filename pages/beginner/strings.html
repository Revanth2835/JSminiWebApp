Strings & Methods

Strings represent text. JavaScript strings come with built-in properties and methods to easily measure length, replace words, change casing, and slice parts out of the text.

Real-world Analogy

Imagine a String as a bead necklace where each character is a bead. String methods are like tools you use on the necklace: `.length` counts the beads, `.toUpperCase()` paints them all bright, and `.slice()` cuts a specific section out for you.

Syntax Overview

Using built-in String methods:

let txt = "Hello World";
console.log(txt.length); // 11
console.log(txt.toUpperCase()); // "HELLO WORLD"
console.log(txt.replace("World", "JS")); // "Hello JS"
