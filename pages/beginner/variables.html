Variables

Variables are named containers used to store data in a program so it can be referenced and manipulated later.

Real-world Analogy

Imagine a variable as a labeled moving box. You write a name on the box (the variable name) and put something inside (the data value). Later, when you need the item, you simply look for the labeled box instead of trying to remember the exact item.

Syntax Overview

In modern JavaScript, there are two primary ways to declare variables:

let age = 25; // Can be reassigned
const name = "Alice"; // Cannot be reassigned
