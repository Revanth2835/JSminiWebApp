Arrays & Methods

An array is a special kind of object used to store multiple values in a single ordered list. Arrays come with powerful built-in methods like map(), filter(), and reduce().

Real-world Analogy

An array is like a bookshelf. You can place many books (values) on it, and each position has an exact index number (starting from 0). Array methods are like librarians who specifically pull out matching books (filter) or translate all titles to another language (map).

Syntax Overview

Declaring and using functional array methods:

const nums = [1, 2, 3];
const doubled = nums.map(n => n * 2); // [2, 4, 6]
const sum = nums.reduce((total, n) => total + n, 0); // 6
